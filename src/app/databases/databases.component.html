Other Aromantic and Asexual Story Databases

There are a number of other databases where stories with aromantic and asexual spectrum characters can be found. These are listed below.

A character based database containing details from books, with information on where the character can be found, what relationships they have, what genre the books are in, and if there are any particular notes and warnings.

The database has not been updated recently so does not contain the most recent releases.

A book/series based database split into genre tranches which are filterable by various means. Contains reasonably detailed description for each entry on the content of the representation and any other relevant information.
